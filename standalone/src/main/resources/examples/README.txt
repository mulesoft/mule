Examples are no longer included out-of-the-box with the Mule distribution. Mule Runtime examples can be found in:

- MuleSoft Documentation - http://www.docs.mulesoft.com
- Anypoint Studio - download from https://www.mulesoft.com/platform/studio
- Anypoint Exchange - https://www.mulesoft.com/exchange

From Anypoint Exchange, you can also access the direct URL for the example that you can download using a utility like wget.
